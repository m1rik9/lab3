## Задание 1

**Условие:**
Реализовать две функции для проверки, является ли переданная последовательность (список, строка или кортеж) палиндромом
**Описание проделанной работы:**
Для проверки достаточно сравнить первый элемент с последним, второй с предпоследним и т.д. Если на каком-то шаге элементы не равны — последовательность не палиндром. Если все пары совпадают или длина последовательности ≤ 1 — это палиндром.

Реализованы две независимые программы, каждая со своей функцией `is_palindrome`. Программы не используют глобальные переменные, все данные передаются через аргументы и локальные переменные.
```python
def is_palindrome(seq):
    if len(seq) <= 1:
        return True
    if seq[0] != seq[-1]:
        return False
    return is_palindrome(seq[1:-1])

# Примеры использования
if __name__ == "__main__":
    print(is_palindrome([1, 2, 3, 2, 1])) 
    print(is_palindrome('spam'))
    print(is_palindrome('radar')) 
    print(is_palindrome((1, 2, 1)))

```

## Задание 2

**Условие:**
Напишите функцию `is_palindrome(seq)`, которая получает последовательность (список, строку или кортеж) и проверяет, является ли она палиндромом. В этой задаче **нельзя использовать рекурсию**.
**Описание проделанной работы:**
Я написал итеративную функцию, которая использует два указателя: один указывает на начало последовательности, другой — на конец. В цикле `while`, пока левый указатель меньше правого, выполняется:  
- Сравниваются элементы на этих позициях. Если они разные — функция сразу возвращает `False`.  
- Если элементы равны, левый указатель сдвигается вправо на один шаг, а правый — влево на один шаг.  
Цикл продолжается, пока указатели не встретятся или не пересекутся. После успешного завершения цикла все пары элементов совпали, значит, последовательность — палиндром, и функция возвращает `True`. 
```python

def is_palindrome(seq):
    left = 0
    right = len(seq) - 1
    while left < right:
        if seq[left] != seq[right]:
            return False
        left += 1
        right -= 1
    return True

# Примеры использования
if __name__ == "__main__":
    print(is_palindrome([1, 2, 3, 2, 1])) 
    print(is_palindrome('spam'))
    print(is_palindrome('radar'))
    print(is_palindrome((1, 2, 1)))

```
